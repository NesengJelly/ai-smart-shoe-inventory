# AI Smart Shoe Inventory & Sales Automation System

Perangkat lunak ini merupakan sistem asisten AI yang dirancang untuk membantu toko sepatu dalam mengotomatisasi pengelolaan stok, penjualan, rekomendasi produk, dan analisis tren pembelian. Sistem ini terintegrasi dengan API Tukupatu (marketplace) dan menggunakan arsitektur berbasis Java/JavaFX dengan database MySQL.
